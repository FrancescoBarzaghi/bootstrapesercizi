A cosa serve bootstrap?

Bootstrap è un potente toolkit frontend ricco di funzionalità. Costruisci qualsiasi cosa, dal prototipo alla produzione, in pochi minuti.

Cosa devi importare per creare una pagina web con bootstrap?

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Utilizzare un CDN per Bootstrap significa includere i file CSS e JavaScript direttamente dal server di un Content Delivery Network.
Questo permette di caricare le pagine web più velocemente e in modo più affidabile, senza dover scaricare i file sul proprio server.

A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

questo permette al tuo sito web di adattarsi automaticamente a schermi di diverse dimensioni, come smartphone e tablet, offrendo una migliore esperienza utente.

he differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina)

Container e container fluid sono classi Bootstrap per definire la larghezza massima di un elemento contenitore.
Container ha una larghezza massima definita, mentre container fluid si estende a tutta la larghezza dello schermo.
Scegli container per layout centrati e container fluid per layout a larghezza piena.
